## 环境

node 15+

```
yarn 或者 npm i 安装依赖
```


## 本地开发启动
```

npm run dev 启动项目
```


## 部署项目
```
yarn build 或者 npm run build 构建静态文件
yarn build 或者 npm run start 部署启动 3000端口
```
